# The Stayease 

**The Description** is a web application that displays a list of accommodations for booking. Interested clients can browse available options and book accommodations. This project uses mock data to simulate booking functionality.

## Key Features
- Displays a list of accommodations
- Users can browse and “book” accommodations (mock data)
- Responsive design for desktop and mobile
- Experimental project for learning UI and interactivity

## Tech Stack
React.js | Tailwind CSS | Git/GitHub

## Project Status
Experimental — core UI and functionality implemented using mock data

## Skills Demonstrated
Frontend development, React component structure, UI design with Tailwind CSS, handling mock data, responsive web design
